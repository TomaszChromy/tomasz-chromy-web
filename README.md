This is my personal portfolio website for freelance web development and junior programming work.

The site focuses on:

- Business websites
- Small e-commerce shops
- Landing pages and conversion-focused pages

The design follows a dark, modern, business-oriented style with a hero banner, sections for clients, services, portfolio, pricing, blog and contact.
